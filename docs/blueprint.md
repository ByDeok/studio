# **App Name**: Golden Wellness

## Core Features:

- Onboarding Flow: Simplified, step-by-step onboarding process tailored for seniors with a focus on ease of use and quick setup (3 minutes).
- Daily Mission Card: Presents 1-3 simple health missions each day, promoting consistent engagement and health habit formation. Completion of missions triggers a rewarding confetti animation to positively reinforce habit creation.
- Health Report Summary: Generates a concise, one-page report of key health data (blood pressure, glucose, steps) suitable for sharing with healthcare providers. Data presented in an easy to read manner.
- Family Connection: Enables family members to monitor the senior's health status through a shared activity feed and receive notifications of important events, providing a supportive and connected care experience.

## Style Guidelines:

- Primary color: Teal (#20c997) provides a sense of trust and reliability, while remaining senior-friendly.
- Background color: Very light teal (#f0f9f5) ensures comfortable contrast and legibility, while staying within the selected light scheme.
- Accent color: Green (#90ee90) highlights key interactive elements to make it easier for Seniors to accomplish tasks within the application.
- Body and headline font: 'PT Sans' (sans-serif) combines a modern look and a little warmth or personality for comfortable reading.
- Use of large buttons (minimum height 56px) for easy interaction, especially for users with dexterity challenges.
- Employ a bottom navigation bar instead of a hamburger menu to provide easy access to the app's main sections.
- Use subtle animations (e.g., confetti) for positive feedback to enhance the user experience, rewarding Seniors when a task has been completed.
